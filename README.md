# CatCovidTracker

Tracking Covid-19 vaccination in the Catalan population.

The code uses the latest vaccination data from the Catalan Department of Health and population data from the Statistical Institute of Catalonia 

Python libraries:
- pandas
- geopandas
- matplotlib
- numpy
- seaborn
